Aplicação de Notas Next.js
Uma aplicação moderna de gerenciamento de notas construída com Next.js, shadcn/ui, Kinde Auth, Prisma e Stripe.
---
Tecnologias Utilizadas
- Next.js  
- shadcn/ui  
- Kinde Auth  
- Prisma  
- Stripe  
- PostgreSQL  
- Tailwind CSS  
- TypeScript  
---
Visão Geral do Sistema
Arquitetura full-stack com Prisma ORM e PostgreSQL. Autenticação via Kinde Auth e assinaturas gerenciadas pelo Stripe.
---
Funcionalidades
- **Autenticação de Usuário**: Sistema seguro de registro e login  
- **Gerenciamento de Notas**: Criar, editar e excluir notas  
- **Sistema de Assinatura**: Integração com Stripe e webhooks  
- **Configurações de Usuário**: Personalização de perfil e cores  
- **Interface Moderna**: UI responsiva com shadcn/ui  
- **Suporte a Temas**: Modo escuro/claro personalizável  
- **Painel de Controle**: Hub centralizado de gerenciamento
